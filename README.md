# My Calculator – La Pascaline

Calculatrice en Python (mode terminal).

Fonctionnalités :
- Opérations : +, -, *, /, ^
- Fonctions unaires : sqrt, fact
- Priorité des opérations respectée
- Gestion des erreurs (division par zéro, entrées invalides)
- Historique des calculs enregistré dans un fichier JSON

Contraintes respectées :
- Pas de eval()
- Pas de module math
